# Semana 12 – Proyecto Final React Mastery (CRUD + Asincronía)  

📌 El objetivo es construir una aplicación que implemente un CRUD completo sobre cualquier temática(ej: tareas, productos, películas, contactos, notas, gastos, etc.), aplicando manejo de estado, asincronía y rutas.

**Fecha de entrega:** Domingo, 14 de Junio de 2026 hasta la media noche

**¿Qué es un CRUD?**

### CRUD representa las que se realizan sobre datos:

* Crear nuevos registros
* Listar o mostrar registros
* Editar registros existentes
* Eliminar registros

**Ejemplo:** Crear una tarea, listar todas las tareas, editar una tarea, elliminar una tarea.

**Stack obligatorio**

* React.js (hooks useState, useEffect, mnejo de asincronía, rutas)
* Vite
* React Router DOM
* CSS / Tailwind / estilos libres  
* Persistencia de datos usando: localStorage, json-server, mockapi.io o Supabase.
